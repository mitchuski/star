## City Key 🗝️

A portable JSON artifact minted at agentprivacy `/city`. Import it on any page to
light your vertices, pour focus, walk the tour, and mint your trace back into it.
Every exported key is stamped with a **κ-label** — `sha256:H(canonical form)`, in
[UOR-ADDR](https://github.com/UOR-Foundation/uor-addr) style — and every import
re-derives and checks it. *Identity is content, not location.* An evolved export
also stamps **`prior`** — the κ of the key it grew from — so keys form a
checkable lineage; a key carrying measured **`figures`** renders its shape from
measurement, not choice. The disclosure ladder: full key → sigil portrait →
⚡ charge pass (proof only) → 🤝 blinded intersection (the common ground only).
